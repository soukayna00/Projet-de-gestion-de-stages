# Projet-de-gestion-de-stages
Application web développée avec Laravel (backend API) et Vue.js (frontend), utilisant une architecture séparée. Le projet inclut une analyse UML complète, la gestion des données (CRUD), l’authentification des utilisateurs, ainsi que l’intégration des pratiques DevOps (Docker, CI/CD) et des tests (unitaires et d’intégration).


## Setup

docker compose up -d --build

http://localhost:8000
